# SDL Game Engine Demo

Demo applications for [SDL Game Engine](https://github.com/linkdd/sdl-game-engine).

# Build

You need ``cmake`` and a compiler supporting the **C++14** standard.

    $ git submodule update --init --recursive
    $ mkdir __build__
    $ cd __build__
    $ cmake ..
    $ make
